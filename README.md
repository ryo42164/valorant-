# VALORANT round win probability estimation

## overview
This project aims to predict round win probability in professional valorant matches.
the long-term goal is to visualize how win probability changes from beggining to end of each round and evaluate the value of individual plays ,including not only explicit events such as kils but also positioning and tactical movements.

## current status
This project is currently in the data preparation and baseline

## Motivation
As mentioned in overview, round outcomes are affected not only kills but also subtle factors. This project explores whether these factors can be estimated and evaluated from broadcast footage.

## data
Raw VOD files,extracted frames,large processed dataset are not included due to storage constraints.
Some sample files are included.
